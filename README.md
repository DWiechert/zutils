# zutils

Zig re-write of uutils.

## Building

Build the project and all utilities with:
```
zig build
```

Run all tests:
```
zig build test --summary all
```

Run individual tests:
```
zig test src/cat.zig
```

## cat

Usage:
```
./zig-out/bin/cat <file 1> <file 2>
```
